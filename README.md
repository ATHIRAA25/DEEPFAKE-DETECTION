 # Deepfake Detection System

## Overview

A deep learning-based system developed to identify whether a video is real or manipulated using DeepFake technology.

## Methodology

1. Upload Video
2. Extract Frames
3. CNN Feature Extraction
4. LSTM Analysis
5. Prediction

## Features

- DeepFake Detection
- Frame Extraction
- Video Analysis
- Prediction Results

## Technologies Used

- Python
- TensorFlow
- CNN
- LSTM
- OpenCV
